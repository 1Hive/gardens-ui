## How to run locally

#### Clone the repository

```
git clone https://github.com/1Hive/gardens-ui.git
```

#### Install dependencies

```
cd gardens-ui
yarn
```

#### Start UI

```
yarn start
```

### Environment variables

Check out the [.env.example file](.env.example)
